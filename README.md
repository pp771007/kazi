## ✨ 功能特色

| | |
|---|---|
| 🔍 **多站聚合搜尋** | 同時搜多個資源站、依片名自動聚合來源；支援排除詞（token 前加 `-`）、一鍵繁→簡 |

| ⏯️ **歷史 / 收藏** | 自動記進度、接續觀看；收藏卡顯示看到第幾集；一鍵檢查更新 |

| 📲 **遠端遙控** | 用手機掃 QR 操控電視盒打字、搜尋、管理站台 |

## ⏯️ 觀看歷史 / 我的收藏

- 看片自動記住進度，下次點同一集從上次時間點接續
- 收藏卡直接顯示「看到第幾集 + 進度」，點下去接續觀看
- 歷史頁可一鍵檢查所有片是否有更新（多了新集數會標記）

## 🔄 帳號同步

搭配網頁版 [**maccms-parser**](https://github.com/pp771007/maccms-parser) 用同一組帳號，觀看歷史 + 收藏就能在手機 / 電視盒 / 網頁之間互通。

- **設定**：設定 → 帳號同步 → 填網頁版網址 + 密碼 → 按「儲存並測試連線」（網址貼根目錄或子頁都行，會自動解析）
- **自動同步**：之後每次進「歷史」或「收藏」頁會自動拉取其他裝置的最新資料；只有同步失敗才會跳提示
- **刪除也同步**：在任一端刪掉歷史 / 收藏，其他裝置同步後也會跟著消失
- 已綁定會顯示「已綁定：暱稱」與伺服器；要換帳號按「解除綁定」

## 🕶️ 無痕模式

點頂列的 **無痕** 按鈕（眼睛劃掉的圖示）即可開啟。開啟後：搜尋不留紀錄、看片不寫入觀看歷史。狀態會記住，App 重開維持上次設定。

## 📲 遠端遙控

<details>
<summary>電視盒用遙控器打字很痛苦？用手機掃 QR 來操控（點開看用法）</summary>

<br>

**開啟**：首頁右上角的 QR Code icon 按一下，會自動啟用並顯示 QR Code。

**手機端**：
1. 跟電視盒連同一個 WiFi
2. 用相機掃 QR（最快），或手動輸入畫面上的網址
3. 手機瀏覽器會開啟一個有四個分頁的網頁：
   - **📱 遠端搜尋**：手機打字、勾要搜的站，送出後電視盒自動跳到搜尋結果
   - **📥 匯入站點**：把另一台裝置的站台設定貼過來匯入（原裝置「匯出站點到剪貼簿」→ 這裡貼上 → 預覽 → 匯入）
   - **⚙️ 站點管理**：新增 / 啟停 / 排序 / 刪除站台
   - **🔄 同步**：用手機打字設定帳號同步的網址 + 密碼

**關閉**：首頁標題旁的「🟢 遠端遙控」膠囊點一下，或回 QR 頁按「停止遠端遙控」。

> ⚠️ **沒有密碼保護** — 同一個 WiFi 上只要拿到網址的人都能進來操控，**只建議在自家 WiFi 使用**。

</details>

## ➕ 站點怎麼來

App **不內建任何站台**。你需要自己提供 MacCMS 規格的 API URL（通常長得像 `https://example.com/api.php/provide/vod/`）。

設定 → 站點管理：
- 手動貼一個 URL 新增
- 「掃描站台」會開 Google 搜尋幫你找候選（不附帶任何站台）
